Element

Element är taggar som används hela tiden, vanliga element är t.ex html, body, h1 och p.

HTML

Hypertext Markup Language eller (HTML) är ett märkspråk för hypertext som är den grundläggande standarden för www (world wide web). Medan HTML är grunden för sidan behövs både CSS (Utseendet) och Javascript (Funktionerna) för en fungerande sida.

ID

ID ger en html tagg en unik attribut som sedan används för att specifikt peka på den idn i en stylesheet.

Class

Class kan antingen vara flera eller bara en specifik tagg. Class används när man vill välja en specifik tagg för att ändra på i ens stylesheet.

Meta

Meta är en prefix som betyder mellan, efter eller bakom.

Responsive

Att ha en responsive sida innebär att den justerar sig efter användarens skärmstorlek och upplösning.

http

,

Hypertext Transfer Protocol är ett kommunikationsprotokoll som används för att flytta sidor till WWW (world wide webb). Den ursprungliga uppgiften för http var att skapa en metod för att överföra HTML-sidor från webbserverarna till webbklienter.

https

Hypertext Transfer Protocol Secure är ett protokoll är krypterad transport med data för HTTP-protokollet.
